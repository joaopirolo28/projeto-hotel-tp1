\doxysubsection{Detailed Description}
Classe para armazenar e validar um valor monetário. 

Armazena o valor internamente como um número inteiro de centavos para evitar problemas de arredondamento com ponto flutuante. O valor deve estar no intervalo de 0,01 a 1.\+000.\+000,00.
