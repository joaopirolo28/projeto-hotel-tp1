\doxysubsection{Detailed Description}
Classe para armazenar e validar um número de telefone. 

O número de telefone deve seguir o formato internacional +DDDDDDDDDDDD, consistindo do caractere \textquotesingle{}+\textquotesingle{} seguido por 12 dígitos.
