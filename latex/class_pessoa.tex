\doxysubsection{Detailed Description}
Classe base que representa uma \doxylink{class_pessoa}{Pessoa}. 

Esta classe serve como alicerce para outras entidades como \doxylink{class_gerente}{Gerente} e Hóspede, contendo atributos comuns a ambos, como nome e email. Os atributos são protegidos para permitir o acesso direto por classes derivadas.
